How-to



  1. Login to Dominos Pizza Malaysia GPS Tracker.
  2. Navigate until you can see your order with blinking current status:

  3. Open browser console (F12 on most browsers) and go to Network tab. Then click on any requests in the list that resembles the order cart id. After that click Headers to get the Authorization Bearer token:


    Make sure that the token is one single line with no line breaks. Edit in text editor if necessary to remove any line breaks and extra spaces. The expected format is Bearer xyz, with one space between Bearer and the actual token.
  4. To get the order CartId url, right click on the request and copy the url:

  5. With the obtained order CartId url and Authorization Bearer token, Track now!
